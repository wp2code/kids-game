- **技术栈与构建**：采用 Vue 3 (Composition API) + TypeScript + Vite 架构，使用 Pinia 进行全局状态管理（`src/stores/game.store.ts`），Vue Router 处理页面路由（`src/router/index.ts`）。
- **核心逻辑分层**：
  - **状态层**：`game.store.ts` 维护游戏阶段（IDLE, PLAYING, FEEDBACK 等）、分数及答题历史。
  - **引擎层**：`useGameEngine` Composable 负责题目随机抽取、干扰项生成及流程控制。
  - **服务层**：`sound-source.service.ts` 优先调用 Freesound API 获取真实音效，失败时降级至 `sound-synth.ts` Web Audio 合成器；`audio-player.ts` 统一管理音频播放生命周期。
  - **视图层**：`App.vue` 监听 Store 中的 `phase` 变化自动驱动路由跳转，实现首页、游戏页与结果页的无缝切换。
- **资源管理**：通过 `vite.config.ts` 配置环境变量注入 API Key，并使用 `@` 别名简化模块引用。
